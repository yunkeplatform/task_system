# 前端项目使用说明

## 如何使用

项目使用了gulp和bower

拉取代码后，先执行 

        npm install
        
再执行

        bower install
        
然后执行

        gulp

## css
这个目录里面存放sass编译过的css样式文件，此时的文件未压缩
## js
* app

  存放项目js文件
 
* lib
 
  需要的组件（大多数为js组件，所以放到了这里）
  
 ## node_module
 项目打包，压缩，前端调试等需要的node组件
 
## sass
 未编译过的.scss 格式的样式表
